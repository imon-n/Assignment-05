Question-1: What are the different ways to select an element in the DOM?

There are five common way to select an element in the DOM. Such as Using getElementById, Using getElementsByClassName, Using getElementsByTagName, Using querySelector and Using querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent?

The difference is innerHtml return all text including html tags, innertext return only all text and textContent is all text contained by an element and all its children.

Question-3: What is event delegation in the DOM?

Event delegation is a design pattern/system of javascipt. its used to manage and handle events on multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a pattern where an event triggered on a element propagations. it also allows parent element to respond to event triggered by their child elements.
